VolvoWrench

This tool is a cross engine fully featured demo analyzing tool. It works with GoldSource and Source-engine demos. Features:

  • Changing font
  • Parsing source engine demos
  • Viewing what's happening in the demos (improved version of netdecode)
  • Global hotkeys

Supported:

  • Half-Life 2

  • Half-Life: Source Deathmatch
